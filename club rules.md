# Dice Combat Game Rules

These rules outline a structured dice-based combat system for role-playing fights. All rolls use a d20 (20-sided die). To roll, type `/roll d20` in the chat. Add modifiers as `/roll d20+<modifier>` or `/roll d20-<modifier>`. The system emphasizes phases to extend fights, with attackers and defenders alternating based on roll outcomes.

## 1. Basic Roll Mechanics

- **Initiative**: Both parties roll a d20 at the start. The highest roll determines who starts as the "attacker" (goes first). Ties are rerolled.
- **Turn Sequence**:
  - The attacker role-plays (RPs) their intended action and rolls.
  - The defender rolls and then RPs the outcome based on the rolls.
- **Winning a Roll**: The highest total (roll + modifiers) wins the action. 
  - If both roll a 1, it's a wash (no effect; reroll or proceed as neutral).
  - If both roll a 20 (before modifiers), the defender wins.
  - There are no "natural 20s" for auto-success beyond standard crit rules (see Section 3). A roll totaling 20 (e.g., 19 +1) is just a 20.
- **Critical Failure (Rolling a 1)**: Counts as an automatic loss for that phase. Skip the current phase and move to the next one. The roller loses initiative if applicable.
- **Ties**: Unless specified otherwise (e.g., in stuns), the defender wins ties.

## 2. Fight Phases

Fights progress through six phases to allow for drawn-out, engaging combat. Phases advance or regress based on roll outcomes. The attacker tries to advance; the defender resists.

1. **Struggle/Grapple/Pin**:
   - Attacker wins: Advance to Phase 2.
   - Attacker loses: Defender becomes the attacker; restart Phase 1.

2. **Strip/Fully Pin**:
   - Attacker wins: Opponent is stripped naked and fully pinned. Advance to Phase 3.
   - Attacker loses: Regress to Phase 1.

3. **Penetration**:
   - Attacker wins: Attacker penetrates the opponent (or equivalent action for bottoms, e.g., handjob, milking, riding). Advance to Phase 4. Defender gains a -1 to all future rolls for each of their failure in this phase (stacks cumulatively).
   - Attacker loses: Regress to Phase 2.

4. **Fuck Check 1**:
   - Attacker wins: Defender becomes fatigued (-1 to all defense rolls indefinitely; stacks for each success in this phase). Advance to Phase 5.
   - Attacker loses: Regress to Phase 3.
   - Example: If Phase 4 is won twice (with a loss in between), defender gets -2 total.

5. **Fuck Check 2**:
   - Attacker wins: Defender gains a permanent -2 to all defense rolls (stacks with Phase 4 debuffs and for each success in this phase). Advance to Phase 6.
   - Attacker loses: Regress to Phase 4.

6. **Climax**:
   - Attacker wins: Attacker wins the fight, having bred/mated the opponent (or equivalent domination/milking for subs).
   - Attacker loses: Regress to Phase 5.

## 3. Critical Hits (Crits)

- **Achieving a Crit**: Final roll total must be 20 or higher (e.g., base roll of 18 +2 modifier = crit). You cannot crit if debuffs prevent reaching 20+.
- **Effect**: Auto-success that can only be beaten by another crit (resulting in a draw). Crits beat non-crits automatically.
- **Exceptions**: Crits do not alter stun outcomes (see Section 5).

## 4. Pre-Fight Modifiers

These are determined before the fight and apply to all rolls unless noted.

- **Size Difference** (based on height; does NOT affect defense rolls):
  - 1-2 feet difference: Larger fighter +1 to attacks; smaller -1 to attacks.
  - More than 2 feet: Larger +2 to attacks; smaller -2 to attacks.

- **Loss Streak** (applies to all rolls; resets fully on a win):
  - 2 losses in a row: -1.
  - 3 losses: -2.
  - 4+ losses: -3.

- **Win Streak** (applies to all rolls):
  - 2 wins in a row: +1.
  - 3 wins: +2.
  - 4+ wins: +3 (considered "champion status").

## 5. Special Moves: Stuns

Stuns are high-risk, high-reward actions usable at any phase via a standard attack roll.

- **Success**: Defender is stunned. Skip the next phase automatically.
  - Example: If Phase was 1, the stun makes it so Phase 2 gets skipped and the game goes to Phase 3.
  - Applies a permanent -2 to the defender's rolls.
- **Failure**: Attacker is vulnerable and regresses:
  - If Attacker was still clothed: Go to Phase 2 (strip/fully pin)..
  - If Attacker was already naked: Go to Phase 3 (penetration).
  - Rolling a 1 auto-fails the stun (regardless of opponent's roll).
  - Failed stuns do NOT apply the -2 debuff to the attacker (to balance risk).
- **Crits on Stuns**: No special effect; outcomes follow standard stun rules.
- **Ties**: Reroll (only move requiring this).
- **Special Interaction with Critical Failure**: If attacker rolls a 1 in Phase 1, defender can attempt a stun as counter. Success skips two phases total (-2 debuff to attacker). Defender's 1 applies both failures to them instead.

## 6. Multi-Person Fights

- **2v1 (Handicap)**:
  - Solo fighter gets +1 to all rolls (adrenaline buff).
  - Solo must beat both opponents' rolls; if either opponent wins, solo's action is nullified (twice the risk).

- **2v2**:
  - Specify targets in RP before rolling.
  - All four RP, then roll.
  - If you target someone not targeting you and win: Their action is nullified in favor of yours.
  - Example: A and B target C; D targets B and wins → B's action nullified; A vs. C proceeds normally.
  - Recommendation: Use tag-team style (2 in, 2 out). Roll to tag or prevent tagging to avoid complexity.

These rules aim for balanced, narrative-driven fights. Adjust for consent and fun in play.
